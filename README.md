# 糖果启动器

### Windows 搜索启动类工具。只做一个功能，启动快捷方式

目标：使用便捷、简单美观、轻便小巧、超低延迟响应

特点：

- 支持小鹤双拼（后续支持全拼及其他双拼）

- 支持首拼、英文首字

- 支持索引 UWP 应用

- 配置快捷方式时，可直接添加整个文件夹

- 搜索结果非常非常跟手

- 软件920 KB，内存已提交占用 150MB 左右

## How to build
1. 克隆当前存储库

2. 在 Visual Studio 中打开解决方案，然后选择 Build
> 仅支持 Windows10 x64

> 需要 .net framework 4.8

> Visual Studio 2022

3. 构建结果位于目录 build 中
